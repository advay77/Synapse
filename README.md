# 🌌 Synapse AI: Neural Meeting Interface

![Synapse Preview](https://github.com/advay77/Synapse/raw/main/client/public/preview.png)

> **The Protocol for Infinite Intelligence.**  
> A high-performance, AI-orchestrated meeting platform designed for seamless real-time cognitive collaboration.

---

## ✨ Features

- **🧠 Synapse AI Co-Pilot**: An integrated AI participant that listens, speaks, and assists during sessions.
- **✨ Premium UI/UX**: Built with a "Mainframe/Cybernetic" aesthetic featuring glassmorphism, neural animations, and a sleek dark theme.
- **📱 Fully Responsive**: Optimized for every device—from desktops to smartphones—maintaining a premium look and feel.
- **🔐 Authorization Protocol**: Secure, room-based entry system with custom "Entity Identity" management.
- **💬 Intelligence Feed**: Live-streaming metadata and message history synced across all participants.
- **⚡ Real-time Orchestration**: Low-latency video, audio, and AI interaction powered by modern Web SDKs.

## 🚀 Tech Stack

- **Frontend**: [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State Management**: [React Hooks](https://react.dev/)
- **Backend/API**: Node.js / Express
- **AI Engine**: Vapi Web SDK for AI Voice & Orchestration
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+)
- npm / yarn / pnpm

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/advay77/Synapse.git
   cd Synapse
   ```

2. **Setup Client**:
   ```bash
   cd client
   npm install
   # Create a .env.local and add your VAPI keys if required
   npm run dev
   ```

3. **Setup Server**:
   ```bash
   cd ../server
   npm install
   node index.js
   ```

## 🛠 Project Structure

- `/client`: Next.js application, UI components, and meeting hooks.
- `/server`: WebSocket/Backend logic for session orchestration.

## 📱 Responsiveness

Synapse has been carefully tuned to provide a desktop-grade experience on mobile. 
- **Adaptive Sidebars**: Intelligence feeds stack on smaller viewports.
- **Optimized Controls**: Action buttons are tailored for touch targets.
- **Dynamic Layout**: The video grid automatically adjusts for 1-n participants.
